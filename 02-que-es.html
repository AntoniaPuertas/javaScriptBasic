¿Qué es JavaScript?

Tradicionalmente

  • Es el lenguaje de la web
  • Añade interactividad a los sitios web
  • Se utiliza para el desarrollo Front End

En la actualidad se utiliza para

  • Desarrollo en Back End
  • Desarrollo de aplicaciones móviles y de escritorio

Usos comunes

  • Validar formularios
  • Reaccionar a eventos del usuario
  • Realizar peticiones al servidor y mostrar los resultados desde un segundo plano

JavaScript es un lenguaje de programación porque tiene:

  • Variables
  • Funciones
  • Estructuras de control

DOM - Document Object Model

Es la estructura de la página HTML

El DOM te permite acceder a las diferentes etiquetas del documento HTML, a sus atributos, propiedades, texto...

Tecnologías basadas en JavaScript

  • React
  • VueJS
  • Angular

JavaScript es la base para muchas otras tecnologías

Si aprendes JavaScript será mucho más fácil aprender React, VueJS, Angular...
