Destructuring de Objetos en JavaScript

El destructuring de objetos es una característica de JavaScript que nos permite extraer propiedades de un objeto y asignarlas a variables de una manera más concisa y legible.

1. Destructuring Básico

El destructuring nos permite "desempaquetar" propiedades de un objeto en variables separadas.

Ejemplo:

const persona = {
  nombre: "Ana",
  edad: 28,
  ciudad: "Madrid"
};

const { nombre, edad } = persona;

console.log(nombre); // Imprime: Ana
console.log(edad); // Imprime: 28

2. Asignando Nombres Diferentes a las Variables

Podemos asignar nombres diferentes a las variables durante el destructuring.

Ejemplo:

const { nombre: nombreCompleto, ciudad: ubicacion } = persona;

console.log(nombreCompleto); // Imprime: Ana
console.log(ubicacion); // Imprime: Madrid

3. Valores por Defecto

Podemos asignar valores por defecto en caso de que la propiedad no exista en el objeto.

Ejemplo:

const { nombre, edad, profesion = "Desconocida" } = persona;

console.log(profesion); // Imprime: Desconocida

4. Destructuring de Objetos Anidados

También podemos hacer destructuring de propiedades en objetos anidados.

Ejemplo:

const estudiante = {
  nombre: "Carlos",
  edad: 20,
  direccion: {
    calle: "Calle Mayor",
    ciudad: "Barcelona"
  }
};

const { nombre, direccion: { ciudad } } = estudiante;

console.log(nombre); // Imprime: Carlos
console.log(ciudad); // Imprime: Barcelona

5. Destructuring en Funciones

El destructuring es muy útil en los parámetros de las funciones para extraer propiedades de objetos pasados como argumentos.

Ejemplo:

function imprimirInfo({ nombre, edad }) {
  console.log(`${nombre} tiene ${edad} años.`);
}

const persona = { nombre: "Luis", edad: 35, ciudad: "Valencia" };
imprimirInfo(persona); // Imprime: Luis tiene 35 años.

6. Uso Práctico del Destructuring

El destructuring es muy útil cuando trabajamos con APIs o grandes objetos de configuración.

Ejemplo: Configuración de una Aplicación

const configuracionApp = {
  servidor: "https://miservidor.com",
  puerto: 3000,
  baseDatos: "miDB",
  opciones: {
    timeout: 5000,
    cache: true
  }
};

const { servidor, puerto, opciones: { timeout } } = configuracionApp;

console.log(`Conectando a ${servidor}:${puerto} con timeout de ${timeout}ms`);

Recuerda: El destructuring de objetos hace que tu código sea más limpio y fácil de leer, especialmente cuando trabajas con objetos complejos o necesitas extraer múltiples propiedades de un objeto.
