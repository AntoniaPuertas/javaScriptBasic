Variables en JavaScript

Variables con let

  • let sustituye a var
  • se declaran igual
  • siguen las mismas reglas para nombrarlas
  • la diferencia está en el alcance de la variable:
    • el alcance (scope) o acceso de let se limita al bloque dónde se declara
    • var se declara ly accede de forma global

Variables con const

  • No se puede inicializar una variable sin valor
  • No se puede modificar su valor
  • const es una constante

Resumen:

Utiliza let a no ser que estés muy seguro de que el valor de la variable no va a cambiar
