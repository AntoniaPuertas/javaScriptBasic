Datos Booleanos y Lógica en JavaScript

En este documento, explicaremos los datos booleanos en JavaScript, su uso en expresiones lógicas y las tablas de verdad para los operadores lógicos básicos.

1. Datos Booleanos

Los datos booleanos solo pueden tener dos valores: true (verdadero) o false (falso). Se utilizan para representar estados binarios y en expresiones lógicas.

Ejemplo Básico:

let esDiaLaboral = true;
let esFinDeSemana = false;
console.log(esDiaLaboral); // Imprime: true
console.log(esFinDeSemana); // Imprime: false

2. Operadores de Comparación

Los operadores de comparación devuelven un valor booleano.

Ejemplos:

console.log(5 > 3); // true (mayor que)
console.log(5 < 3); // false (menor que)
console.log(5 >= 5); // true (mayor o igual que)
console.log(5 <= 4); // false (menor o igual que)
console.log(5 === 5); // true (igualdad estricta)
console.log(5 !== 4); // true (desigualdad estricta)

3. Operadores Lógicos

JavaScript tiene tres operadores lógicos principales: AND (&&), OR (||), y NOT (!).

Ejemplos:

let x = true;
let y = false;
console.log(x && y); // false (AND)
console.log(x || y); // true (OR)
console.log(!x); // false (NOT)

5. Ejemplo Práctico

Veamos un ejemplo que combina booleanos y operadores lógicos en una situación cotidiana.

Determinar si alguien puede conducir:

let edad = 19;
let tieneLicencia = true;

let puedeConducir = edad >= 18 && tieneLicencia;
console.log("¿Puede conducir?", puedeConducir); // Imprime: ¿Puede conducir? true

Recuerda: Los datos booleanos y la lógica son fundamentales en la programación, especialmente en estructuras de control como condicionales y bucles.
