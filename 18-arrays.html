Arrays en JavaScript

Los arrays en JavaScript son estructuras de datos que nos permiten almacenar múltiples elementos en una sola variable. Son muy útiles para manejar listas de información.

1. Creación de Arrays

Hay varias formas de crear un array en JavaScript:

Ejemplo: Creación de Arrays

// Usando corchetes
let frutas = ['manzana', 'banana', 'naranja'];

// Usando el constructor Array
let numeros = new Array(1, 2, 3, 4, 5);

console.log(frutas); // Imprime: ['manzana', 'banana', 'naranja']
console.log(numeros); // Imprime: [1, 2, 3, 4, 5]

2. Acceso a Elementos del Array

Podemos acceder a elementos individuales del array usando su índice (posición). Los índices comienzan en 0.

Ejemplo: Acceso a Elementos

let colores = ['rojo', 'verde', 'azul'];

console.log(colores[0]); // Imprime: rojo
console.log(colores[2]); // Imprime: azul

3. Modificación de Arrays

Podemos modificar elementos existentes o añadir nuevos elementos a un array.

Ejemplo: Modificación y Adición

let animales = ['perro', 'gato', 'conejo'];

animales[1] = 'pájaro'; // Modificar un elemento
animales.push('pez'); // Añadir al final

console.log(animales);
// Imprime: ['perro', 'pájaro', 'conejo', 'pez']

4. Longitud de un Array

Podemos conocer cuántos elementos tiene un array usando la propiedad length.

Ejemplo: Longitud del Array

let dias = ['lunes', 'martes', 'miércoles', 'jueves', 'viernes'];
console.log(dias.length); // Imprime: 5

5. Métodos Comunes de Arrays

JavaScript proporciona varios métodos útiles para trabajar con arrays:

Ejemplo: Métodos de Arrays

let números = [1, 2, 3, 4, 5];

números.push(6); // Añade al final
números.unshift(0); // Añade al principio
let último = números.pop(); // Elimina y devuelve el último
let primero = números.shift(); // Elimina y devuelve el primero

console.log(números); // Imprime: [1, 2, 3, 4, 5]
console.log(último); // Imprime: 6
console.log(primero); // Imprime: 0

6. Iteración sobre Arrays

Podemos recorrer todos los elementos de un array usando bucles o métodos específicos.

Ejemplo: Iteración con for

let frutas = ['manzana', 'banana', 'cereza'];

for (let i = 0; i < frutas.length; i++) {
  console.log(frutas[i]);
}
// Imprime:
// manzana
// banana
// cereza

Ejemplo: Iteración con forEach

frutas.forEach(function(fruta) {
  console.log(fruta);
});
// Imprime lo mismo que el ejemplo anterior

Recuerda: Los arrays son estructuras de datos muy versátiles en JavaScript. Te permiten organizar y manipular colecciones de datos de manera eficiente. Practicar con arrays es esencial para dominar la programación en JavaScript.
