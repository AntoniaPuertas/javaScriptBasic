Variables en JavaScript

  • ¿Qué es una variable?
  • ¿Qué son las palabras reservadas?

Características de JavaScript

  • El punto y coma al final de cada instrucción es opcional
  • No hay que indicar el tipo de dato al declarar / inicializar una variable
  • Es de tipo dinámico:
    • Puedes cambiar el tipo de dato de una variable

Reglas para los nombres de las variables

  • Pueden contener letras, números, guión medio y guión bajo
  • No pueden comenzar por un número
  • No pueden comenzar por un guión medio
  • JavaScript es CaseSensitive: Distingue entre mayúsculas y minúsculas

Estilos de nomenclatura para las variables

  • nombre_alumno (underscore)
  • nombreAlumno (Camelcase)
  • NombreAlumno (Pascal Case) (Se utiliza para crear clases)
