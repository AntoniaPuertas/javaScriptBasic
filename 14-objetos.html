Objetos en JavaScript para Principiantes

Los objetos en JavaScript son estructuras que nos permiten agrupar datos relacionados y funcionalidades. Pensemos en ellos como contenedores que pueden almacenar diferentes tipos de información.

1. Creación de Objetos

Hay varias formas de crear objetos en JavaScript. La más común es usando llaves {}.

Ejemplo:

let persona = {
  nombre: "Ana",
  edad: 28,
  ciudad: "Madrid"
};

console.log(persona); // Imprime: {nombre: "Ana", edad: 28, ciudad: "Madrid"}

2. Propiedades de Objetos

Las propiedades son los datos almacenados en un objeto. Cada propiedad tiene un nombre (clave) y un valor.

Accediendo a Propiedades:

console.log(persona.nombre); // Imprime: Ana
console.log(persona["edad"]); // Imprime: 28

3. Modificando Objetos

Podemos cambiar los valores de las propiedades existentes o añadir nuevas propiedades a un objeto.

Ejemplo:

persona.edad = 29; // Cambiando una propiedad existente
persona.profesion = "Ingeniera"; // Añadiendo una nueva propiedad

console.log(persona);
// Imprime: {nombre: "Ana", edad: 29, ciudad: "Madrid", profesion: "Ingeniera"}

4. Métodos de Objetos

Los métodos son funciones que pertenecen a un objeto. Nos permiten realizar acciones con los datos del objeto.

Ejemplo:

let coche = {
  marca: "Toyota",
  modelo: "Corolla",
  año: 2020,
  arrancar: function() {
    console.log("El coche está arrancando");
  }
};

coche.arrancar(); // Imprime: El coche está arrancando

5. Objetos Anidados

Los objetos pueden contener otros objetos como propiedades, permitiendo estructuras de datos más complejas.

Ejemplo:

let escuela = {
  nombre: "Colegio San Juan",
  direccion: {
    calle: "Calle Mayor",
    numero: 123,
    ciudad: "Barcelona"
  },
  numEstudiantes: 500
};

console.log(escuela.direccion.ciudad); // Imprime: Barcelona

6. Uso Práctico de Objetos

Los objetos son muy útiles para organizar información relacionada y crear estructuras de datos complejas.

Ejemplo: Lista de Contactos

let listaContactos = [
  {nombre: "Juan", telefono: "123-456-7890", email: "juan@email.com"},
  {nombre: "María", telefono: "098-765-4321", email: "maria@email.com"},
  {nombre: "Carlos", telefono: "555-555-5555", email: "carlos@email.com"}
];

console.log(listaContactos[1].nombre); // Imprime: María

Recuerda: Los objetos son una parte fundamental de JavaScript y se utilizan extensamente en el desarrollo web moderno. Practicar con objetos te ayudará a escribir código más organizado y eficiente.
