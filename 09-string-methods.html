Métodos de String en JavaScript

JavaScript proporciona varios métodos útiles para trabajar con strings. Aquí se presentan algunos de los más comunes:

1. length

Devuelve la longitud de la cadena.

Ejemplo:

let str = "Hola Mundo";
console.log(str.length); // Imprime: 10

2. toLowerCase()

Convierte la cadena a minúsculas.

Ejemplo:

let str = "HOLA Mundo";
console.log(str.toLowerCase()); // Imprime: "hola mundo"

3. toUpperCase()

Convierte la cadena a mayúsculas.

Ejemplo:

let str = "Hola Mundo";
console.log(str.toUpperCase()); // Imprime: "HOLA MUNDO"

4. trim()

Elimina los espacios en blanco al principio y al final de la cadena.

Ejemplo:

let str = " Hola Mundo ";
console.log(str.trim()); // Imprime: "Hola Mundo"

5. charAt()

Devuelve el carácter en la posición especificada.

Ejemplo:

let str = "Hola Mundo";
console.log(str.charAt(0)); // Imprime: "H"

6. indexOf()

Devuelve la posición de la primera ocurrencia de un valor especificado en una cadena.

Ejemplo:

let str = "Hola Mundo";
console.log(str.indexOf("Mundo")); // Imprime: 5

*Si el valor devuelto es -1 significa que el valor espedificado no se encuentra en la cadena

7. slice()

Extrae una parte de una cadena y la devuelve como una nueva cadena.

Ejemplo:

let str = "Hola Mundo";
console.log(str.slice(0, 4)); // Imprime: "Hola"

8. split()

Divide una cadena en un array de subcadenas.

Ejemplo:

let str = "Hola Mundo";
console.log(str.split(" ")); // Imprime: ["Hola", "Mundo"]

9. replace()

Reemplaza un valor especificado con otro valor en una cadena.

Ejemplo:

let str = "Hola Mundo";
console.log(str.replace("Mundo", "JavaScript")); // Imprime: "Hola JavaScript"

10. includes()

Determina si una cadena contiene los caracteres de una cadena especificada.

Ejemplo:

let str = "Hola Mundo";
console.log(str.includes("Mundo")); // Imprime: true

Estos son solo algunos de los métodos disponibles para strings en JavaScript. Cada uno de estos métodos puede ser muy útil en diferentes situaciones al trabajar con cadenas de texto.
