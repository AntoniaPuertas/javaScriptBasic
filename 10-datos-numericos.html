Métodos para Números en JavaScript

JavaScript proporciona varios métodos útiles para trabajar con números. Aquí se presentan algunos de los más comunes:

1. toFixed()

Devuelve una representación de cadena de un número con un número específico de decimales.

Ejemplo:

let num = 3.14159;
console.log(num.toFixed(2)); // Imprime: "3.14"

2. toPrecision()

Devuelve una representación de cadena de un número con una longitud específica.

Ejemplo:

let num = 3.14159;
console.log(num.toPrecision(3)); // Imprime: "3.14"

3. toString()

Devuelve una representación de cadena de un número en una base específica.

Ejemplo:

let num = 255;
console.log(num.toString(16)); // Imprime: "ff" (hexadecimal)

4. parseInt()

Analiza una cadena y devuelve un entero.

Ejemplo:

let str = "42";
console.log(parseInt(str)); // Imprime: 42

5. parseFloat()

Analiza una cadena y devuelve un número de punto flotante.

Ejemplo:

let str = "3.14";
console.log(parseFloat(str)); // Imprime: 3.14

6. Math.round()

Redondea un número al entero más cercano.

Ejemplo:

let num = 3.6;
console.log(Math.round(num)); // Imprime: 4

7. Math.floor()

Redondea un número hacia abajo al entero más cercano.

Ejemplo:

let num = 3.6;
console.log(Math.floor(num)); // Imprime: 3

8. Math.ceil()

Redondea un número hacia arriba al entero más cercano.

Ejemplo:

let num = 3.2;
console.log(Math.ceil(num)); // Imprime: 4

9. Math.abs()

Devuelve el valor absoluto de un número.

Ejemplo:

let num = -5;
console.log(Math.abs(num)); // Imprime: 5

10. Math.random()

Devuelve un número pseudoaleatorio entre 0 (inclusive) y 1 (exclusivo).

Ejemplos:

// Número entre 0 y 1
console.log(Math.random()); // Imprime: un número entre 0 y 1, por ejemplo, 0.7231321432131

// Número entre 1 y 20
let numeroAleatorio = Math.floor(Math.random() * 20) + 1;
console.log(numeroAleatorio); // Imprime: un número entero entre 1 y 20

11. Math.min()

Devuelve el número más pequeño de los argumentos proporcionados.

Ejemplo:

console.log(Math.min(5, 10, 3, 8, 1)); // Imprime: 1

12. Math.max()

Devuelve el número más grande de los argumentos proporcionados.

Ejemplo:

console.log(Math.max(5, 10, 3, 8, 1)); // Imprime: 10

Estos son solo algunos de los métodos disponibles para trabajar con números en JavaScript. Cada uno de estos métodos puede ser muy útil en diferentes situaciones al realizar operaciones matemáticas o manipular valores numéricos.
