Concatenación de Strings y Template Strings en JavaScript

En este documento, explicaremos cómo unir strings (cadenas de texto) en JavaScript usando dos métodos: concatenación tradicional y template strings.

1. Concatenación de Strings

La concatenación es una forma de unir dos o más strings. En JavaScript, podemos usar el operador '+' para concatenar strings.

Ejemplo Básico:

let nombre = "Juan";
let saludo = "Hola, " + nombre + "!";
console.log(saludo); // Imprime: Hola, Juan!

Concatenación con Números:

Cuando concatenamos strings con números, JavaScript convierte automáticamente los números a strings.

let edad = 25;
let mensaje = "Tengo " + edad + " años.";
console.log(mensaje); // Imprime: Tengo 25 años.

2. Template Strings (String Literals)

Los template strings son una forma más moderna y flexible de crear strings. Se utilizan con comillas invertidas (`) y permiten insertar expresiones directamente en el string.

Ejemplo Básico:

let nombre = "María";
let saludo = `Hola, ${nombre}!`;
console.log(saludo); // Imprime: Hola, María!

Expresiones en Template Strings:

Puedes incluir cualquier expresión JavaScript válida dentro de ${}.

let a = 5;
let b = 10;
console.log(`La suma de ${a} y ${b} es ${a + b}.`);
// Imprime: La suma de 5 y 10 es 15.

Strings Multilinea:

Los template strings también permiten crear strings de varias líneas de forma sencilla.

let poema = `
Roses are red,
Violets are blue,
JavaScript is fun,
And so are you!`;
console.log(poema);

Recuerda: Los template strings son más flexibles y legibles, especialmente cuando trabajas con strings complejos o que incluyen variables. Sin embargo, la concatenación tradicional sigue siendo útil en muchas situaciones.
