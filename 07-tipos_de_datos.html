Tipos de Datos en JavaScript

JavaScript es un lenguaje de tipado dinámico, lo que significa que las variables pueden contener diferentes tipos de datos. Estos son los principales tipos de datos en JavaScript:

En programación cuando se habla de tipo de dato se refiere al tipo de valor que va a contener la variable

1. Number

Representa tanto números enteros como de punto flotante.

Ejemplo: let edad = 25; o let precio = 19.99;

2. String

Representa texto y se define con comillas simples o dobles.

Ejemplo: let nombre = "Juan"; o let apellido = 'Pérez';

3. Boolean

Representa un valor verdadero o falso.

Ejemplo: let esMayor = true; o let tienePermiso = false;

4. Undefined

Representa una variable que ha sido declarada pero no ha sido asignada con un valor.

Ejemplo: let variableSinDefinir;

5. Null

Representa la ausencia intencional de cualquier valor u objeto.

Ejemplo: let valorNulo = null;

6. Object

Representa una colección de datos relacionados.

Ejemplo: let persona = {nombre: "Ana", edad: 30};

7. Array

Es un tipo especial de objeto que representa una lista ordenada de valores.

Ejemplo: let colores = ["rojo", "verde", "azul"];

8. Symbol (ES6)

Representa un identificador único.

Ejemplo: let id = Symbol("id");

9. BigInt (ES11)

Representa números enteros muy grandes.

Ejemplo: let numeroGrande = 1234567890123456789012345678901234567890n;

Recuerda que puedes usar el operador typeof para verificar el tipo de dato de una variable en JavaScript.
